# 🎓 Student Life Guide Website

A responsive web project built using **HTML and CSS** to help students stay productive, balanced, and motivated.

---

## 📌 Project Overview

This project was created as part of a task to build a **responsive webpage layout using CSS Flexbox and Grid**.

Instead of creating a random website, I chose to design something meaningful — a **Student Life Guide** that provides helpful tips on:
- 📚 Study habits  
- ⏰ Time management  
- 💙 Wellness  

---

## 🎯 Objectives

- Build a responsive layout using **Flexbox and CSS**
- Structure content using clean and semantic HTML
- Ensure the website adapts across:
  - 💻 Desktop  
  - 📱 Tablet  
  - 📱 Mobile  

---

## 🛠️ Technologies Used

- HTML 
- CSS3  
- Flexbox  
- CSS  
- Media Queries  

---

## 📱 Responsive Design

The website is fully responsive and adapts to different screen sizes:

- **Desktop:** Full layout with multi-column sections  
- **Tablet:** Adjusted spacing and 2-column layouts  
- **Mobile:** Stacked layout for easy readability  

---

## 🧱 Key Features

- Clean and structured layout  
- Hero section with background image overlay  
- Navigation bar with interactive buttons   
- Smooth hover effects and transitions  
- Mobile-friendly design
